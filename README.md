## Getting Started

![image](https://github.com/MatheusInacio/react-python/assets/4120362/520b6da3-a864-44ea-9cb4-dc1fa7c152a1)


Run Frontend
```bash
cd frontend
npm run dev
```

Install Backend
```
cd backend
pip install -r requirements.txt
```

Run Backend
```
cd backend
.\.venv\Scripts\activate
python manage.py runserver
```
